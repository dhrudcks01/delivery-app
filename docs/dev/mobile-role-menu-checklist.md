# T-0524 권한 신청/승인 메뉴 내정보 하위 메뉴화 체크리스트

## 점검 대상
- 앱 로그인 후 `내정보` 탭
- 역할별 계정: USER / DRIVER / OPS_ADMIN / SYS_ADMIN

## 시나리오 1: USER 메뉴 노출
1. USER 계정으로 로그인 후 `내정보 > 권한 신청/승인` 진입
2. 노출 메뉴 확인

기대 결과:
- `DRIVER 권한 신청` 메뉴만 노출된다.
- `OPS_ADMIN 권한 신청` 메뉴는 노출되지 않는다.

## 시나리오 2: DRIVER 메뉴 노출
1. DRIVER 계정으로 로그인 후 `내정보 > 권한 신청/승인` 진입

기대 결과:
- `OPS_ADMIN 권한 신청` 메뉴가 노출되지 않는다.
- 안내 문구만 노출된다.

## 시나리오 3: OPS_ADMIN 메뉴 노출
1. OPS_ADMIN 계정으로 로그인 후 `내정보 > 권한 신청/승인` 진입

기대 결과:
- `OPS_ADMIN 권한 신청` 메뉴는 노출되지 않는다.
- 필요 시 `SYS_ADMIN 권한 신청` 메뉴만 노출된다.

## 시나리오 4: SYS_ADMIN 승인 접근
1. SYS_ADMIN 계정으로 로그인 후 `내정보 > 권한 신청/승인` 진입
2. `권한 승인/부여 화면 열기` 선택

기대 결과:
- OPS_ADMIN/SYS_ADMIN 신청 승인 화면으로 진입한다.
- 요청/이용내역 탭에서는 승인 메뉴 직접 진입 대신 안내 화면이 노출된다.
