# T-0523 USER 탭 정책 수동 점검 체크리스트

## 점검 대상
- 앱 로그인 사용자: USER 단일 권한 계정
- 플랫폼: Android/iOS (Expo)

## 시나리오 1: 신청 탭 직접 진입 차단 확인
1. USER 계정 로그인 후 하단 `신청` 탭 진입
2. 안내 문구가 표시되고, 수거 신청 입력 폼이 바로 노출되지 않는지 확인

기대 결과:
- `내정보` 탭에서 신청을 시작하라는 정책 안내가 보인다.
- 신청 폼 직접 노출/중복 진입이 없다.

## 시나리오 2: 내정보에서 신청 진입 확인
1. 하단 `내정보` 탭 진입
2. `기사 신청하기` 버튼 선택
3. 신청 화면에서 주소 검색/상세주소/연락처 입력 후 요청 생성

기대 결과:
- 신청 화면 진입이 `내정보` 경로로만 가능하다.
- 요청 생성 성공 시 오류 없이 처리된다.

## 시나리오 3: 이용내역 탭 확인
1. 하단 `이용내역` 탭 진입
2. 신청 목록/상세가 로드되는지 확인
3. 상태값(REQUESTED/ASSIGNED/COMPLETED 등)과 생성/수정 시각 확인

기대 결과:
- 신청 내용 및 이용 상태를 `이용내역` 탭에서 확인할 수 있다.
- 항목 선택 시 상세 정보가 정상 표시된다.

## 시나리오 4: 로그아웃 위치 확인
1. `홈`, `신청`, `이용내역` 탭에서 로그아웃 버튼 존재 여부 확인
2. `내정보` 탭에서 로그아웃 버튼 확인 및 동작 확인

기대 결과:
- 로그아웃 버튼은 `내정보` 탭에만 존재한다.
- 로그아웃 수행 시 로그인 화면으로 이동한다.
