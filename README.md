# Delivery MVP

This repository is the workspace for the Delivery MVP.
It is organized into:

- `server/` — backend API (Spring Boot 3 / Java 17 / Gradle)
- `mobile/` — mobile app (Expo + TypeScript)
- `docs/` — project documentation and ticket tracking

## Local development constraints

- No Docker requirement for local development.
- Server targets Spring Boot 3, Java 17, Gradle.
- Mobile targets Expo (React Native) with TypeScript.
